Quick Start Guide
=================

1. Run the Evolution Simulator
2. Click "📂 Load CodeBase" button
3. Navigate to: sample_codebase/
4. Click "Open"
5. Wait for loading to complete
6. Watch as 4 modules appear:
   - StringUtils.java
   - ArrayUtils.java  
   - Calculator.java
   - DESIGN.txt
7. Click "▶ Run One Generation" to evolve
8. Observe fitness scores and graphs updating

Each file becomes a "digital organism" that evolves!
